# Vim Mapping
### **-- Navigate within insert mode**
- `<C-b>` : beginning of line
- `<C-e>` : end of line
- `<C-h>` : move left
- `<C-l>` : move right
- `<C-j>` : move down
- `<C-k>` : move up


- `<C-s>` : Save file
- 

### **--Comment**
- `<leader>/` : comment in both modes (normal, insert)


### **--Tab**
- `<S-b>` : New tab (new buffer)
- `<tab>` : goto next tab
- `<S-tab>` : goto prev tab
- `<leader>x` : close tab

### **--Git**
- `<leader>gt` : Git status
- `<leader>gc` : Git commit
